---
title: Payin Quotes
description: Methods for creating and managing payin quotes
---

The Payin Quotes resource provides methods to create quotes for fiat-to-crypto conversions and retrieve foreign exchange rates.

## create

Create a quote for a payin transaction. The quote locks in an exchange rate and fee structure for a limited time.

```typescript
const response = await blindpay.payins.quotes.create({
  blockchain_wallet_id: 'bw_1234567890',
  currency_type: 'sender',
  payment_method: 'pix',
  request_amount: 1000,
  token: 'USDC',
  cover_fees: true,
  partner_fee_id: null,
  is_otc: false
});

if (response.error) {
  console.error(response.error.message);
} else {
  console.log(response.data.id);
  console.log(`Rate: ${response.data.commercial_quotation}`);
  console.log(`Expires at: ${new Date(response.data.expires_at * 1000)}`);
}
```

### Parameters

<ParamField body="blockchain_wallet_id" type="string" required>
  ID of the blockchain wallet that will receive the stablecoins.
</ParamField>

<ParamField body="currency_type" type="'sender' | 'receiver'" required>
  Whether the `request_amount` is in fiat currency (`sender`) or stablecoin (`receiver`).
</ParamField>

<ParamField body="payment_method" type="'ach' | 'wire' | 'pix' | 'spei' | 'transfers' | 'pse'" required>
  Payment method to be used for the transaction.
  - `ach`: Automated Clearing House (US)
  - `wire`: Wire transfer (US)
  - `pix`: Instant payment (Brazil)
  - `spei`: SPEI transfer (Mexico)
  - `transfers`: Bank transfer (Argentina)
  - `pse`: PSE payment (Colombia)
</ParamField>

<ParamField body="request_amount" type="number" required>
  The amount to quote. Interpreted based on `currency_type`.
</ParamField>

<ParamField body="token" type="'USDC' | 'USDT' | 'USDB'" required>
  The stablecoin token to receive.
</ParamField>

<ParamField body="cover_fees" type="boolean" required>
  Whether fees should be covered by the sender (`true`) or deducted from the receiver amount (`false`).
</ParamField>

<ParamField body="partner_fee_id" type="string | null" required>
  ID of the partner fee configuration to apply, or `null` for no partner fee.
</ParamField>

<ParamField body="is_otc" type="boolean" optional>
  Whether this is an over-the-counter transaction.
</ParamField>

<ParamField body="payer_rules" type="PayerRules" optional>
  Rules and restrictions for the payer.

  <ParamField body="pix_allowed_tax_ids" type="string[]" optional>
    Allowed tax IDs for PIX payments.
  </ParamField>

  <ParamField body="transfers_allowed_tax_id" type="string" optional>
    Allowed tax ID for transfers (Argentina).
  </ParamField>

  <ParamField body="pse_allowed_tax_ids" type="string[]" optional>
    Allowed tax IDs for PSE payments.
  </ParamField>

  <ParamField body="pse_full_name" type="string" optional>
    Full name for PSE payments.
  </ParamField>

  <ParamField body="pse_document_type" type="'CC' | 'NIT'" optional>
    Document type for PSE payments (CC: Citizenship Card, NIT: Tax ID).
  </ParamField>

  <ParamField body="pse_document_number" type="string" optional>
    Document number for PSE payments.
  </ParamField>

  <ParamField body="pse_email" type="string" optional>
    Email for PSE payments.
  </ParamField>

  <ParamField body="pse_phone" type="string" optional>
    Phone number for PSE payments.
  </ParamField>

  <ParamField body="pse_bank_code" type="string" optional>
    Bank code for PSE payments.
  </ParamField>
</ParamField>

### Response

<ResponseField name="data" type="CreatePayinQuoteResponse">
  The created quote details.

  <ResponseField name="id" type="string">
    Unique identifier for the quote.
  </ResponseField>

  <ResponseField name="expires_at" type="number">
    Unix timestamp (in seconds) when the quote expires.
  </ResponseField>

  <ResponseField name="commercial_quotation" type="number">
    The commercial exchange rate for the transaction.
  </ResponseField>

  <ResponseField name="blindpay_quotation" type="number">
    The BlindPay exchange rate including fees.
  </ResponseField>

  <ResponseField name="receiver_amount" type="number">
    Amount that will be received in stablecoin.
  </ResponseField>

  <ResponseField name="sender_amount" type="number">
    Amount that needs to be sent in fiat currency.
  </ResponseField>

  <ResponseField name="partner_fee_amount" type="number | null" optional>
    Partner fee amount if applicable.
  </ResponseField>

  <ResponseField name="flat_fee" type="number">
    Flat fee charged for the transaction.
  </ResponseField>

  <ResponseField name="is_otc" type="boolean" optional>
    Whether this is an OTC transaction.
  </ResponseField>
</ResponseField>

## getFxRate

Retrieve the current foreign exchange rate between currencies without creating a quote.

```typescript
const response = await blindpay.payins.quotes.getFxRate({
  currency_type: 'sender',
  from: 'BRL',
  to: 'USDC',
  request_amount: 5000
});

if (response.error) {
  console.error(response.error.message);
} else {
  console.log(`Commercial rate: ${response.data.commercial_quotation}`);
  console.log(`BlindPay rate: ${response.data.blindpay_quotation}`);
  console.log(`Result amount: ${response.data.result_amount}`);
}
```

### Parameters

<ParamField body="currency_type" type="'sender' | 'receiver'" required>
  Whether the `request_amount` is in the source currency (`sender`) or destination currency (`receiver`).
</ParamField>

<ParamField body="from" type="'USDC' | 'USDT' | 'USDB' | 'BRL' | 'USD' | 'MXN' | 'COP' | 'ARS'" required>
  Source currency for the conversion.
</ParamField>

<ParamField body="to" type="'USDC' | 'USDT' | 'USDB' | 'BRL' | 'USD' | 'MXN' | 'COP' | 'ARS'" required>
  Destination currency for the conversion.
</ParamField>

<ParamField body="request_amount" type="number" required>
  The amount to convert. Interpreted based on `currency_type`.
</ParamField>

### Response

<ResponseField name="data" type="GetPayinFxRateResponse">
  Foreign exchange rate information.

  <ResponseField name="commercial_quotation" type="number">
    The commercial market exchange rate.
  </ResponseField>

  <ResponseField name="blindpay_quotation" type="number">
    The BlindPay exchange rate including fees.
  </ResponseField>

  <ResponseField name="result_amount" type="number">
    The resulting amount after conversion.
  </ResponseField>

  <ResponseField name="instance_flat_fee" type="number">
    Flat fee charged by your instance.
  </ResponseField>

  <ResponseField name="instance_percentage_fee" type="number">
    Percentage fee charged by your instance.
  </ResponseField>
</ResponseField>
